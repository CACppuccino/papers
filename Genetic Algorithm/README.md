*注1：在量子资讯科学中，量子位元（又称为Q位元，量子比特），是量子信息的计量单位。传统电脑使用0和1，量子电脑也是使用0跟1，但与之不同的是，其0与1可同时计算。古典系统中，一个位元在同一时间，不是0，就是1，但量子位元是0和1的量子叠加。这是量子电脑计算的特性。(wikipedia)*
